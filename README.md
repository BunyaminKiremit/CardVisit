# Android Card Visit
<p> 
Bu uygulama XMl ve Jsoup kullanarak Türkiye Cumhuriyet Merkez Bankası'ndan alınan döviz kurlarını gösteren bir android uygulamasıdır.
              
### Uygulamada Kullanılan Özellikler:
                    
Name  | Version
------------- | -------------
Jsoup  | 1.16.1
Kotlin  | 1.8
</p>
